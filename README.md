# Xiaomi Gaming Mouse Lite

## Specifications  
Chip: Holtek HT32F5XXXX  
- Flash: 32kb `0x00000000 - 0x00007FFF`  
- SRAM: 4kb `0x20000000 - 0x20000FFF`  
- Boot ROM: 4kb `0x1FF00000`  
Sensor: PixArt PAW3327

## Toolkit  
Setup: `pip install hid`  
Start: `python toolkit.py` (With sudo on Linux)
